# 🚀 零基础 30 分钟快速上手 DJ

完全不会 DJ？没问题！按照这个指南，30 分钟后你就能完成第一次混音！

---

## ⏱️ 时间规划

| 时间 | 内容 |
|------|------|
| 0-5 分钟 | 认识设备 + 安装 DJUCED |
| 5-10 分钟 | 生成第一首练习曲 + 导入 |
| 10-15 分钟 | 基础操作：播放、音量、耳机 |
| 15-20 分钟 | 对拍 + 第一次混音 |
| 20-30 分钟 | EQ 基础 + 自由练习 |

---

## 0-5 分钟：认识设备

### 你的 Hercules DJControl Inpulse 200 MK2

只需要记住这几个最重要的部分：

**左侧（Deck A）：**
- 🎛️ **大转盘**：搓盘、找歌
- ▶️ **播放/暂停按钮**：开始/停止
- 🎚️ **音量推子**：控制声音大小
- 🎧 **Cue 按钮**：耳机监听

**中间（Mixer）：**
- ↔️ **横推**：在两首歌之间切换
- 🔊 **主音量**：控制总音量
- 🎧 **耳机音量**：控制耳机声音

**右侧（Deck B）：**
- 和左侧一模一样！

### 安装 DJUCED（如果还没安装）

1. 访问 https://www.djuced.com/download
2. 下载 Windows 版本
3. 安装 + 连接控制器

---

## 5-10 分钟：生成练习曲

### 第一步：启动 AI 生成器

双击运行：
```
start.ps1
```

或者在 PowerShell 中：
```powershell
.\start.ps1
```

### 第二步：生成练习曲

选择菜单：
1. 选择 **"1. 🎵 使用云端 API (MiniMax) 生成"**
2. 选择 **"1. 基本生成 (自定义)"**
3. 输入：
   - 曲目创意：`新手练习，清晰底鼓`
   - 音乐风格：`House`
   - BPM：`120`
   - 器乐曲目：`Y`
   - 自动播放：`Y`

等待生成完成...

### 第三步：在 DJUCED 中导入

1. 打开 DJUCED
2. 左侧找到文件夹浏览器
3. 找到 `08_exports/dj_ready/`
4. 把生成的 MP3 拖到 Deck A

---

## 10-15 分钟：基础操作

### 播放和暂停

1. 按 Deck A 的 **播放/暂停按钮**
2. 音乐响了！
3. 再按一次暂停

### 调整音量

1. 慢慢向上推 Deck A 的 **音量推子**
2. 听声音变大
3. 调整到舒服的音量

### 耳机监听（重要！）

1. 插上耳机
2. 按 Deck A 的 **Cue 按钮**
3. 调整 **耳机音量旋钮**
4. 只有你能听到！

**为什么重要？**
- 准备下一首歌时，听众听不到
- 专业 DJ 都是这么做的！

---

## 15-20 分钟：第一次混音！

### 准备第二首歌

1. 再生成一首练习曲（和第一首 BPM 一样）
2. 拖到 Deck B

### 步骤：

1. **Deck A 播放中**：Deck A 播放，音量推子在上面
2. **Deck B 准备**：
   - 按 Deck B 的 **Cue 按钮**（耳机监听）
   - 按 Deck B 的 **播放按钮**
   - 按 Deck B 的 **SYNC 按钮**（自动对齐速度！）
3. **开始混音**：
   - 慢慢向上推 Deck B 的 **音量推子**
   - 同时慢慢向下拉 Deck A 的 **音量推子**
   - 或者用 **横推** 从左慢慢推到右
4. **完成！** 🎉

你刚刚完成了第一次 DJ 混音！

---

## 20-30 分钟：EQ 基础

### 什么是 EQ？

三个旋钮：
- **Low（左）**：低音（底鼓、贝斯）
- **Mid（中）**：中音（人声）
- **High（右）**：高音（镲片）

### "低切低"技巧（最常用！）

1. Deck B 的 **Low** 旋钮完全转到左边（切掉低音）
2. 慢慢推起 Deck B 的音量
3. 同时把 Deck A 的 **Low** 慢慢切掉
4. 同时把 Deck B 的 **Low** 慢慢打开
5. 完全拉下 Deck A 的音量

**结果：** 完美的混音，没有浑浊的低音！
