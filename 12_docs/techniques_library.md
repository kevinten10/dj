## 基础技巧

### 1. 节拍识别与数拍

#### 4/4 拍结构
```
1 (强) - 2 (弱) - 3 (次强) - 4 (弱)
●     - ●     - ●      - ●
```

**练习方法**：
- 听音乐时数 1-2-3-4
- 重点关注底鼓（Kick）通常在 1 和 3
- 拍手（Clap/Snare）通常在 2 和 4
- 在波形图上寻找规律的峰值

### 2. Beatgrid 校准

**步骤**：
1. 在波形图找到第一个底鼓
2. 将 Beatgrid 第一条线对齐该点
3. 播放检查后续鼓点
4. 微调 BPM 使所有鼓点对齐

**常见问题**：
- AI 生成的曲目可能有 tempo drift（速度漂移）
- 必要时分段调整 Beatgrid
- 关闭 Snap to grid 进行微调

### 3. 对拍（Beatmatching）

#### 使用 SYNC 辅助
1. 加载两首曲目
2. 一首播放，一首暂停
3. 按 SYNC 自动对齐
4. 观察 Pitch 变化

#### 手动对拍
1. 耳机监听待混音的曲目
2. 调整 Pitch 使两首速度接近
3. 在 Cue 点同时播放
4. 微调 Pitch 保持同步

**练习要点**：
- 先从相同 BPM 开始
- 逐步增加难度到 ±5 BPM
- 练习听辨速度差异
- 保持同步至少 30 秒

## 混音技巧

### 1. EQ（均衡器）基础

#### 三频段 EQ
- **Low（低频）**：底鼓、贝斯，20-250Hz
- **Mid（中频）**：人声、乐器，250Hz-4kHz
- **High（高频）**：镲片、Hi-Hat，4kHz-20kHz

#### 低频避让（Frequency Masking）
```
曲目 A（播放中）    曲目 B（待混音）
Low: 0dB    →    Low: -∞dB (完全切掉)
Mid: 0dB          Mid: 0dB
High: 0dB         High: 0dB
```

### 2. EQ 过渡技巧

#### Bass Swap（低切低）
1. 曲目 A 播放中，曲目 B 准备好
2. 曲目 B 的 Low 完全切掉 (-∞)
3. 慢慢推起曲目 B 的 fader
4. 同时慢慢降低曲目 A 的 Low
5. 同时提升曲目 B 的 Low
6. 完全拉下曲目 A 的 fader

#### 渐进式过渡
```
时间 →
曲目 A: Low ↓  Mid →  High ↓
曲目 B: Low ↑  Mid →  High ↑
```

### 3. 结构混音

#### Hot Cue 设置建议
- **Cue 1**: Intro 开始
- **Cue 2**: First Drop
- **Cue 3**: Breakdown
- **Cue 4**: Outro 开始

#### 常见混音点
- **Intro → Intro**：适合开场
- **Verse → Drop**：能量提升
- **Drop → Drop**：保持能量
- **Breakdown → Build**：戏剧性过渡
- **Outro → Intro**：平滑过渡

#### 8/16/32 拍乐句
```
[8拍]  [8拍]  [8拍]  [8拍]
 Intro   Verse   Build   Drop
```

**混音时机**：通常在 32 拍的边界处开始混音

## 进阶技巧

### 1. Looping（循环）

#### 基础循环
- **4 拍循环**：适合快速填充
- **8 拍循环**：标准长度
- **16 拍循环**：更长的循环段落

#### 循环过渡
1. 在曲目 A 的 Drop 开始 8 拍循环
2. 引入曲目 B
3. 当曲目 B 到达 Drop 时
4. 停止循环，切换到曲目 B

#### 半拍循环效果
- 创建紧张感
- 适合 Build-up 阶段
- 配合 Filter 效果更佳

### 2. Harmonic Mixing（和声混音）

#### Camelot Wheel
```
大调 (Major) → 1B-12B
小调 (Minor) → 1A-12A
```

**兼容 Key**：
- 相同数字相邻字母（1A → 1B）：相对大小调转换
- 相邻数字相同字母（1A → 2A）：完美五度
- 数字 ±1，字母相同：能量提升
- 数字 +7（或 -5）：相对调

#### Energy Boost 技巧
1. 从 5A → 12A → 7A → 2A...
2. 每次提升能量
3. 保持和声兼容性

### 3. Set 构建

#### 能量曲线规划
```
时间 →
能量
  ↑
  ●     ●
 ● ●   ● ●
●   ● ●   ●
●     ●     ●
────────────────→
开场    高潮    收尾
```

#### Set 结构示例
- **0-15 分钟**：开场，低能量，110-120 BPM
- **15-30 分钟**：建立，能量提升，120-126 BPM
- **30-45 分钟**：高潮，最高能量，126-130 BPM
- **45-60 分钟**：维持，保持热度
- **60-75 分钟**：收尾，能量下降，120-115 BPM

## 效果器应用

### 1. Filter（滤波器）

#### LPF（低通滤波器）
- 切掉高频
- 创造"闷"的效果
- 适合 Build-up 阶段
- 逐渐打开创造紧张感

#### HPF（高通滤波器）
- 切掉低频
- 创造"薄"的效果
- 适合 Intro/Outro
- 平滑过渡

#### Filter Sweep
1. LPF 慢慢关闭到最低
2. 等待几拍
3. 快速打开
4. 配合 Drop

### 2. Reverb（混响）

**应用场景**：
- Breakdown 时增加空间感
- 过渡时的尾音效果
- 创造氛围营造

**技巧**：
- 短混响：自然感
- 长混响：空间感
- 配合 Filter 使用

### 3. Delay（延迟）

**类型**：
- **1/4 拍延迟**：基础效果
- **1/8 拍延迟**：更快的节奏
- **1/16 拍延迟**：密集效果
- **Ping Pong**：左右立体声

**应用**：
- 人声效果
- 过渡填充
- Build-up 阶段

### 4. 效果器组合

#### Build-up 效果链
```
1. LPF 逐渐关闭
2. Reverb 增加
3. Delay 1/8 拍
4. 快速打开 LPF
5. Drop！
```

#### 过渡效果
```
1. 曲目 A: HPF 逐渐打开
2. 同时 Reverb 增加
3. 曲目 B: LPF 逐渐打开
4. 完全切换
```

## 搓盘技巧

### 1. Baby Scratch

**步骤**：
1. 将唱针（转盘）放在样本开始处
2. 手向前推 → 快速拉回
3. 保持节奏：推-拉-推-拉
4. 配合节拍

**练习**：
- 先从慢速开始
- 逐渐加快速度
- 保持节奏稳定

### 2. Forward Scratch

**技巧**：
- 只向前推
- 不往回拉
- 更自然的声音
- 适合填充使用

### 3. Chirp Scratch

**步骤**：
1. 推转盘时关闭推子
2. 拉转盘时打开推子
3. 创造"唧唧"声

### 4. Transformer Scratch

**技巧**：
- 快速开关推子
- 配合转盘移动
- 创造断续效果
- 节奏感强
