专为零基础用户打造的完整教程！

## 📦 第一步：认识你的设备

### 设备概览

Hercules DJControl Inpulse 200 MK2 是一款完美的入门级 DJ 控制器！

```
┌─────────────────────────────────────────────────────────────┐
│  █████████████████████████████████████████████████████████  │
│  █┌──────┐          ┌──────┐          ┌──────┐          █ │
│  │ Deck A│          │ Mixer│          │ Deck B│          │ │
│  └──────┘          └──────┘          └──────┘          │ │
└─────────────────────────────────────────────────────────────┘
```

### 左侧 (Deck A) 详解

| 部件 | 功能 |
|------|------|
| **转盘 (Jog Wheel)** | 搓盘、搜索曲目、调整速度 |
| **播放/暂停按钮** | 播放或暂停曲目 |
| **Cue 按钮** | 设置和触发提示点 |
| **同步 (SYNC) 按钮** | 自动对齐两首曲目的节拍 |
| **音调滑块 (Pitch Fader)** | 调整曲目速度 (±10%) |
| **4 个打击垫** | Hot Cues / 循环 / 采样 |
| **音量推子** | 控制 Deck A 的音量 |
| **耳机监听按钮** | 只在耳机中听 Deck A |

### 中间 (Mixer) 详解

| 部件 | 功能 |
|------|------|
| **横推 (Crossfader)** | 在 Deck A 和 Deck B 之间平滑过渡 |
| **EQ 旋钮 (Low/Mid/High)** | 调整低音、中音、高音 |
| **主音量旋钮** | 控制总输出音量 |
| **耳机音量旋钮** | 控制耳机音量 |
| **Cue Mix 旋钮** | 混合耳机信号和主输出信号 |

### 右侧 (Deck B) 详解

与左侧完全相同，对应 Deck B！

## 💻 第二步：安装 DJUCED

### 1. 下载 DJUCED

访问官方网站：
```
https://www.djuced.com/download
```

选择：
- **Windows** 或 **macOS**
- **DJUCED 6.5 或更高版本**（推荐）

### 2. 安装步骤

1. 运行下载的安装程序
2. 按照提示点击"下一步"
3. 选择安装位置（默认即可）
4. 等待安装完成
5. 连接你的 Hercules 控制器

### 3. 首次启动

1. 用 USB 线连接控制器到电脑
2. 启动 DJUCED
3. 等待控制器被识别（左下角会显示）
4. 恭喜！你已经准备好了！

## 🎮 第四步：基础操作（5分钟学会）

### 1. 播放和暂停

**操作：**
- 按 Deck A 的 **播放/暂停按钮**
- 音乐开始播放！
- 再按一次暂停

### 2. 调整音量

**操作：**
- 慢慢向上推 Deck A 的 **音量推子**
- 听音乐声音变大
- 向下拉声音变小
- 调整到舒适的音量

### 3. 使用耳机监听

**操作：**
1. 插上耳机
2. 按 Deck A 的 **耳机监听按钮**（Cue）
3. 调整 **耳机音量旋钮**
4. 只有耳机能听到，主输出不响！

**为什么要用耳机？**
- 准备下一首要播放的歌
- 听众听不到你在准备什么
- 专业 DJ 的必备技能！

### 4. 调整速度

**操作：**
- 移动 **音调滑块 (Pitch Fader)**
- 向上：音乐变快
- 向下：音乐变慢
- 中间：原始速度

## 🎯 第五步：第一首歌到第二首歌（你的第一次混音！）

### 准备工作

1. 在 **Deck A** 播放一首歌曲
2. 在 **Deck B** 加载另一首歌曲（风格相似，BPM 接近）

### 步骤详解

#### 1. 耳机准备第二首歌
```
Deck A (正在播放)       Deck B (准备中)
     ↓                        ↓
  主输出播放              耳机监听
  (听众听到)           (只有你听到)
```

- 按 Deck B 的 **耳机监听按钮**
- 你会在耳机里听到 Deck B
- 按 Deck B 的 **播放按钮**

#### 2. 对拍（使用 SYNC）
- 按 Deck B 的 **SYNC 按钮**
- 两首歌的速度会自动对齐！
- 神奇吧？

#### 3. 开始混音
- 慢慢向上推 Deck B 的 **音量推子**
- 同时慢慢向下拉 Deck A 的 **音量推子**
- 用 **横推 (Crossfader)** 也可以，从左慢慢推到右
- 恭喜！你完成了第一次混音！

## 🎛️ 第六步：EQ 的魔力

### 什么是 EQ？

EQ = 均衡器，让你调整音乐的不同频率：

- **Low (低频)**：底鼓、贝斯（咚咚咚）
- **Mid (中频)**：人声、主要乐器
- **High (高频)**：镲片、Hi-Hat（次次次）

### 基础 EQ 混音技巧

**"低切低" (Bass Swap) - 最常用的技巧！**

```
Deck A (播放中)    Deck B (待混音)
  Low: 0dB    →    Low: -∞ (完全切掉)
  Mid: 0dB          Mid: 0dB
  High: 0dB         High: 0dB
```

**步骤：**
1. 在 Deck B 中，把 **Low** 旋钮完全转到左边（-∞）
2. 慢慢推起 Deck B 的音量
3. 同时把 Deck A 的 **Low** 慢慢切掉
4. 同时把 Deck B 的 **Low** 慢慢打开
5. 完全拉下 Deck A 的音量
6. 完美！没有浑浊的低音！

## 🔥 第七步：Hot Cues（提示点）

### 什么是 Hot Cues？

Hot Cues 是你可以快速跳转到的标记点！

### 设置 Hot Cues

1. 播放歌曲
2. 在想要标记的位置按一个 **打击垫**
3. 打击垫会亮起来！
4. 随时按这个打击垫，就会跳回这个位置！

### 推荐的 4 个 Cue 点

| Cue | 位置 | 用途 |
|-----|------|------|
| **1** | Intro 开始 | 从开头播放 |
| **2** | First Drop | 最嗨的部分 |
| **3** | Breakdown | 安静的部分 |
| **4** | Outro 开始 | 准备结束 |

## ⚡ 快速参考表

### 紧急情况

| 问题 | 解决方法 |
|------|----------|
| 没声音 | 检查音量推子、主音量、USB连接 |
| 耳机没声音 | 按 Cue 按钮、检查耳机音量 |
| 控制器没反应 | 重启 DJUCED、重新插拔 USB |

### 键盘快捷键（DJUCED）

| 按键 | 功能 |
|------|------|
| 空格 | 播放/暂停 |
| ← → | 搜索 |
| F1-F4 | Hot Cues 1-4 |
| 鼠标滚轮 | 缩放波形 |
